Each chip is not an encyclopedia entry. It is a working unit. It has a theorem. It has three or four Python programs that actually implement the mathematics. It has couplings — the chips it needs and the chips it feeds — forming a web where pulling any thread pulls others. Every chip is a door into the network, not a page in a book.
